# Local Chat Messenger

A simple local chat messenger using UNIX domain sockets.
It supports both TCP and UDP communication and uses Faker to generate fake responses.

## Features
- TCP client/server communication
- UDP client/server communication
- Fake data generation using Faker
- Local communication using UNIX domain sockets

## Requirements
- Python 3
- Faker
